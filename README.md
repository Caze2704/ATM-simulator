🏧 ATM Simulator

Este projeto é um simulador simples de caixa eletrônico (ATM) desenvolvido utilizando HTML, CSS e JavaScript.
Ele permite simular operações básicas de um caixa eletrônico, como inserir valores e realizar saques, com validações e interação dinâmica na interface.

Este projeto foi criado como prática após a conclusão do meu curso de JavaScript, com o objetivo de aplicar conceitos fundamentais da linguagem no desenvolvimento de uma aplicação web interativa.

🚀 Tecnologias utilizadas

HTML5

CSS3

JavaScript (Vanilla JS)

🎯 Objetivo do projeto

O objetivo deste projeto foi praticar:

Manipulação do DOM

Eventos em JavaScript

Estruturas condicionais

Validação de dados

Interação entre HTML, CSS e JavaScript

Além disso, o projeto faz parte da minha evolução como desenvolvedor e do meu portfólio de estudos.
